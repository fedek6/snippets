# in operator

<LiveExample link="https://jsfiddle.net/fedek6/1mvrhgq7/" />

This operator returns `true` if left operand exists in right operand (object).

```js
const testObject = {
	foo: true
};

console.log("foo" in testObject); // true
console.log("bar" in testObject); // false
```
